# Podcast Format Reference

## Explore with Tony (Travel)
- **Length**: 3-6 minutes (450-900 words)
- **Tone**: Personal, storytelling, inspiring
- **Structure**: Hook → Story → Resources → Close
- **Style**: First-person narratives with sensory details, practical recommendations

## Solo Law Club (Solo Lawyers)
- **Length**: 3-12 minutes (450-1800 words)
- **Tone**: Conversational, pragmatic, educational
- **Structure**: Problem → Framework → Example → Action
- **Style**: Direct address to solo practitioners, numbered frameworks, concrete advice

## 832 Weekends (Parenting)
- **Length**: 3-6 minutes (450-900 words)
- **Tone**: Supportive, vulnerable, reflective
- **Structure**: Moment → Reflection → Connection → Close
- **Style**: Honest, present-tense learning, no advice-giving, emotional precision
